# Flashycardy Docs

Welcome to the documentation site for **Flashycardy**, a full-stack flashcard app built on Next.js 16, Drizzle, Clerk, and the Vercel AI SDK.

This site is intentionally minimal for now — it ships as a placeholder to validate the monorepo + Vercel deployment of `apps/docs`. Content will be backfilled in follow-up work.

## What you'll find here

- **Getting Started** — clone, install, and run locally.
- **Architecture** — how the monorepo and the web app are organized.

## Source

The Flashycardy app and these docs live in the same monorepo. The web app is at `apps/web`; this docs site is at `apps/docs`.
