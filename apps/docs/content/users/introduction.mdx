# Welcome to Flashycardy

Flashycardy helps you learn anything by creating flashcard decks, adding questions and answers, and studying them until the material sticks. Your progress is saved automatically so you can see how you improve over time.

Flashycardy is available on the **web**, as a **Chrome extension**, and on **iOS**. Sign in once with the same account to access your decks everywhere.

## What you can do

| Task | Where to start |
|------|----------------|
| Create and organize decks | [Dashboard](/users/features/decks) — click **New Deck** |
| Add flashcards | Open a deck → **Add Card** |
| Study | Open a deck → **Study** |
| Review past sessions | **Analytics** on the dashboard |
| Change language | User menu → **Settings** |
| Upgrade your plan | **Pricing** or **View Plans** when you hit a limit |

## New here?

Follow the [Getting started](/users/getting-started) guide to create your first deck and complete a study session in a few minutes.
