# Welcome to Flashycardy

Flashycardy helps you learn anything by creating flashcard decks, adding questions and answers, and studying them until the material sticks. Your progress is saved automatically so you can see how you improve over time.

Flashycardy is available on the **web**, as a **Chrome extension**, and on **iOS**. Sign in once with the same account to access your decks everywhere.

## What you can do

| Task | Where to start |
|------|----------------|
| Create and organize decks | [Decks](/users/features/decks) — click **New Deck** |
| Add flashcards | [Cards](/users/features/cards) — open a deck → **Add Card** |
| Study | [Study sessions](/users/features/study-sessions) — open a deck → **Study** |
| Review past sessions | [Analytics](/users/features/analytics) |
| Change language | [Language settings](/users/features/language-settings) |
| Upgrade your plan | [Billing & plans](/users/billing) or [Upgrade to Pro](/users/workflows/upgrade-to-pro) |

## Pro features

With **Pro**, you can:

- Create **unlimited decks** (Free: 3 decks)
- [Generate with AI](/users/features/ai-flashcard-generation) — 20 cards from a deck description
- [Import from document](/users/features/document-deck-import) — new deck from PDF/DOCX/PPTX (web)
- [Generate from web page](/users/features/generate-from-page) — new deck from a page (Chrome extension)
