# Import a document deck

Turn a PDF, Word file, or PowerPoint into a ready-to-study deck in one step — **web app only**, **Pro** required.

## Before you begin

- Pro plan with document deck generation — [Upgrade to Pro](/users/workflows/upgrade-to-pro) if needed
- File under **10 MB**, format `.pdf`, `.docx`, or `.pptx`
- Room for a new deck (Free: under 3 decks)

## Steps

1. Sign in on the [web app](/users/platforms/web).
2. Click **New Deck** on the Dashboard.
3. Select the **Document** tab.
4. Drag your file into the upload area, or click **choose file**.
5. Confirm the selected filename appears.
6. Click **Generate deck**.
7. Wait for processing — you are redirected to the new deck with **20 generated cards**.

## Expected outcome

A new deck with AI-generated name, description, and cards. Review cards, delete weak ones, add manual cards, then [Study a deck](/users/workflows/study-a-deck).

## Troubleshooting

| Problem | Solution |
|---------|----------|
| No Document tab | Upgrade to Pro |
| Wrong file type | Export as PDF, DOCX, or PPTX |
| Error reading file | Try a clearer export; avoid scanned images without OCR |
| AI error | [AI generation failed](/users/troubleshooting/ai-generation-failed) |
