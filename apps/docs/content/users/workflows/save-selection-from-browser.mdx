# Save selection from browser

Quickly capture a snippet from any web page as a flashcard **front** text using the **Chrome extension**.

## When to use it

Use this for one-off facts, definitions, or quotes — faster than opening the web app and typing. For a full deck from a page, use [Generate from web page](/users/features/generate-from-page) (Pro) instead.

## Requirements

- [Chrome extension](/users/platforms/chrome-extension) installed
- Signed in to Flashycardy in the extension

## Steps

1. Select text on a web page (highlight with your mouse).
2. **Right-click** the selection.
3. Choose **Save selection to FlashyCardy** from the context menu.
4. The side panel opens on the **new card** screen with the selection pre-filled as **Front**.
5. Choose a deck (or create flow as prompted), enter the **Back**, and save.

## Expected outcome

A new card is added to your chosen deck. The card appears on the web and iOS when synced.

## Limitations

- Captures **selected text only** — not a full auto-generated deck
- Does not work on pages where Chrome blocks scripting (browser internal pages, some PDF views)
- Requires the extension's context menu permission
