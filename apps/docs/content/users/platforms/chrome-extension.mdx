# Chrome extension

The Flashycardy **Chrome extension** opens in a **side panel** so you can manage decks and study without leaving the page you are reading.

## Install

Install from the Chrome Web Store (when published) or load an unpacked build for development. You need a Flashycardy account — [Sign up](/users/getting-started) on the web if you are new.

## Sign in

1. Click the Flashycardy icon in Chrome's toolbar.
2. The **side panel** opens.
3. Click **Sign in** or **Sign up** if prompted.
4. If you are already signed in on the **web app** at the same origin, your session may sync when you reopen the panel.

## What you can do

| Feature | In extension |
|---------|----------------|
| View and manage decks | Yes |
| Add, edit, delete cards | Yes |
| Study sessions | Yes |
| Analytics | Yes |
| Settings (language) | Yes |
| **Generate from web page** | Yes (Pro) — [Generate from web page](/users/features/generate-from-page) |
| **Save selection** | Yes — context menu |
| Import from PDF/DOCX | No — use [web app](/users/platforms/web) |
| Upgrade / billing | Opens **Pricing** on the web in a new tab |

## Generate from a page

With Pro, use **Generate from this page** in the panel header while viewing an article. See [Generate from web page](/users/features/generate-from-page).

## Save selected text

1. Select text on a web page.
2. Right-click → **Save selection to FlashyCardy**.
3. The side panel opens to add a card with the selection as the **front**.

See [Save selection from browser](/users/workflows/save-selection-from-browser).

## Expected outcome

Decks and study data match your account on the web and iOS. Changes sync through the Flashycardy API.

## Troubleshooting

- **Decks out of date** — Close and reopen the panel; confirm you use the same account as the web app
