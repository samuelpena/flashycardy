# iOS app

The Flashycardy **iOS app** brings decks, study, and analytics to iPhone and iPad with a native SwiftUI interface.

## Install

Download from the App Store when available. For local development builds, use Xcode — see developer docs (not covered here).

## Sign in

1. Open Flashycardy.
2. On the landing screen, tap **Sign in** or **Sign up**.
3. Complete Clerk authentication in the sheet.
4. After sign-in, you see the main tabs: **Dashboard**, **Analytics**, **Settings**.

## What you can do

| Feature | On iOS |
|---------|--------|
| View, create, edit, delete decks | Yes |
| Manage cards | Yes |
| Study sessions | Yes |
| Analytics | Yes |
| Language (Settings tab) | Yes |
| **Generate with AI** on existing deck | Yes (Pro) — [AI flashcard generation](/users/features/ai-flashcard-generation) |
| Import from document | No — use [web app](/users/platforms/web) |
| Generate from web page | No — use [Chrome extension](/users/platforms/chrome-extension) |
| Upgrade to Pro | Opens **Pricing** in Safari (no in-app purchase in app v1) |

## Expected outcome

Your decks stay in sync with the web and extension when signed in with the same account.

## Troubleshooting

- **Sign-in problems** — [Sign-in issues](/users/troubleshooting/sign-in-issues)
- **Missing decks** — Confirm the same email as on web; pull to refresh on Dashboard
- **Pro features blocked** — Upgrade on web **Pricing** page, then retry

See [Sync across devices](/users/troubleshooting/sync-across-devices).
