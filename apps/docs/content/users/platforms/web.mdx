# Web app

The Flashycardy **web app** is the full experience in your browser — dashboard, deck management, study mode, analytics, and billing.

## Sign in

1. Open the Flashycardy site.
2. Click **Sign in** or **Sign up** on the home page (modal — you stay on the same page).
3. After sign-in, you land on the **Dashboard**.

## Main areas

| Area | How to open |
|------|-------------|
| **Dashboard** | Default after sign-in — all your decks |
| **Deck detail** | Click a deck card |
| **Study** | **Study** button on deck detail |
| **Analytics** | **Analytics** link on Dashboard header |
| **Pricing** | **Pricing** in header or upgrade dialogs |
| **Settings** | Profile picture → **Settings** (language) |
| **Account** | Profile picture → **Manage account** |

## Web-only Pro features

- **Import from document** — **New Deck** → **Document** tab ([Import from document](/users/features/document-deck-import))

Features shared with other platforms: manual decks/cards, study, analytics, [AI flashcard generation](/users/features/ai-flashcard-generation).

## Tips

- Use **Sort** on the Dashboard to order decks by date or name.
- The **X/3 decks used** banner appears on the Free plan — see [Billing & plans](/users/billing).
- First visit may show the [Onboarding tour](/users/features/onboarding-tour).
