# Import from document

Create a **new deck** from a file — Flashycardy extracts text, picks a title and description, and generates **20 flashcards** automatically.

## When to use it

Use document import when you already have study material in a file (lecture notes, slides, handouts) and want a deck without manual entry.

## Requirements

- **Pro plan** with document deck generation enabled
- Supported formats: **PDF**, **Word (.docx)**, **PowerPoint (.pptx)**
- Maximum file size: **10 MB**
- Subject to the same **3-deck limit** on Free as manual deck creation (Pro unlimited decks removes this)

## Where it works

| Platform | Supported |
|----------|-----------|
| **Web app** | Yes — **Document** tab in **New Deck** dialog |
| **Chrome extension** | No — use [Generate from web page](/users/features/generate-from-page) instead |
| **iOS** | No |

## Steps (web)

1. On the Dashboard, click **New Deck**.
2. Open the **Document** tab (visible on Pro accounts).
3. Drag and drop a file or click **choose file**.
4. Click **Generate deck**.
5. Flashycardy creates the deck and redirects you to the new deck page with generated cards.

## Expected outcome

A new deck with an AI-chosen name, description, and 20 cards. You can rename the deck, edit cards, or [study](/users/workflows/study-a-deck) immediately.

## Troubleshooting

| Issue | What to try |
|-------|-------------|
| No **Document** tab | Upgrade to Pro — [Billing & plans](/users/billing) |
| Unsupported file type | Use `.pdf`, `.docx`, or `.pptx` only |
| File too large | Split or export a smaller file (under 10 MB) |
| Could not read text | Try a different export format or clearer scan |

See also [AI generation failed](/users/troubleshooting/ai-generation-failed).
