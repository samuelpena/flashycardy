# AI flashcard generation

**Generate with AI** adds **20 new flashcards** to an existing deck based on the deck's **name** and **description**. Flashycardy uses AI to draft question-and-answer pairs that match your topic.

## When to use it

Use AI generation when you have described a topic in the deck description but do not want to type every card manually. It complements [manual cards](/users/features/cards) — generated cards are **appended** to the deck, not replaced.

## Requirements

- **Pro plan** with AI flashcard generation enabled
- The deck must have a **non-empty description** (AI needs context about the topic)

Free users see an upgrade prompt instead of the working button. See [Billing & plans](/users/billing).

## Steps (web & iOS)

### Web

1. Open a deck and add a **Description** if the deck does not have one ([Edit deck](/users/features/decks)).
2. On the deck detail page, click **Generate with AI**.
3. Wait while cards are generated (this may take a few seconds).
4. Refresh the page or scroll the card list — **20 new cards** appear at the end.

If the button is disabled, hover for a tooltip: you need a deck description first.

### iOS

1. Open a deck with a description.
2. Tap **Generate with AI** on the deck screen.
3. New cards appear when generation completes.

### Chrome extension

1. Open a deck in the side panel.
2. Tap **Generate with AI** (Pro required).
3. New cards load in the deck view.

## Expected outcome

Twenty new flashcards are added to the deck. You can edit or delete any card afterward. Study as usual — see [Study sessions](/users/features/study-sessions).
