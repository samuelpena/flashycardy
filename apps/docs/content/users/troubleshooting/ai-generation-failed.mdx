# AI generation failed

If **Generate with AI**, **Generate deck** from a document, or **Generate from this page** fails, try the steps below.

## Common causes

| Cause | What to do |
|-------|------------|
| **Not on Pro** | [Upgrade to Pro](/users/workflows/upgrade-to-pro) |
| **Deck has no description** (Generate with AI) | Edit deck → add **Description** → retry |
| **Unsupported or empty file** (document import) | Use PDF/DOCX/PPTX under 10 MB with readable text |
| **Page too short** (extension) | Use a longer article or more selected text |
| **Deck limit** (Free) | Delete a deck or upgrade — [Deck limit reached](/users/troubleshooting/deck-limit-reached) |
| **Temporary server error** | Wait and retry; check you are still signed in |

## Steps to recover

1. **Confirm Pro status** — open **Pricing** or **Manage account** and verify subscription is active.
2. **Refresh the app** — reload the web page or reopen the extension panel.
3. **Retry with simpler input** — shorter description, smaller file, or different page.
4. **Check network** — AI generation requires an internet connection.

## Expected outcome

On success, you see new cards or a new deck. On repeated failure with Pro active, try again later or contact support with the time and action you attempted.
