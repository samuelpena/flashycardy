## `apps/web` (primary)

| Path | Purpose |
|------|---------|
| `src/app/` | Next.js App Router pages and `/api` route handlers |
| `src/actions/` | Server Actions (web-only mutations) |
| `src/db/queries/` | **All** Drizzle query helpers |
| `src/db/schema/` | Postgres schema (source of truth) |
| `src/lib/mcp/` | MCP auth, tool registration, runners |
| `src/components/` | Web-specific UI |
| `tests/` | Playwright E2E |

Public REST API: `src/app/api/`. MCP: `src/app/api/mcp/route.ts`.

## `apps/docs`

| Path | Purpose |
|------|---------|
| `content/` | MDX pages (`users/`, `developers/`, `api/`) |
| `app/layout.tsx` | Nextra theme + navbar |
| `vercel.json` | Docs-only ignore build step |

## `apps/extension`

| Path | Purpose |
|------|---------|
| `src/entrypoints/` | sidepanel, popup, background, auth-callback |
| `src/screens/` | HashRouter screens |
| `src/lib/api-provider.tsx` | `@flashycardy/api-client` + Clerk token |

## `apps/ios`

| Path | Purpose |
|------|---------|
| `Flashycardy/Flashycardy/` | SwiftUI app sources |
| `Flashycardy/Flashycardy/Services/` | REST API client + services |
| `Config/Secrets.xcconfig` | Clerk + API URL (gitignored) |

## Shared packages

Import via workspace protocol: `"@flashycardy/ui": "workspace:*"`.

## Turborepo tasks

Defined in `turbo.json`:

| Task | Depends on | Notes |
|------|------------|-------|
| `build` | `^build` | `.next/`, `.output/`, `dist/` outputs |
| `lint` | ‚Äî | ESLint per package |
| `test:unit` | `^build` | Vitest |
| `test:e2e` | `^build`, `build` | Playwright; not cached |
| `dev` | ‚Äî | Persistent; not cached |

Run filtered tasks from root, e.g. `pnpm build:docs` ‚Üí `turbo run build --filter=@flashycardy/docs`.

## Cursor rules

Project conventions live in `.cursor/rules/` (data access, auth, shadcn, testing, MCP parity, docs). Summarized in [Coding standards](/developers/coding-standards).
