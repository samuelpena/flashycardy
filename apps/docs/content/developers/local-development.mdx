# Local development

Set up the Flashycardy monorepo on your machine.

## Prerequisites

- **Node.js 22** (matches CI)
- **pnpm 10** (`corepack enable` or install globally)
- **Neon** PostgreSQL database ([neon.tech](https://neon.tech))
- **Clerk** application with Billing enabled ([clerk.com](https://clerk.com))
- **OpenAI** API key (AI flashcard generation)

Optional:

- **Chrome** — extension development
- **Xcode 16+** — iOS development

## 1. Install dependencies

```bash
git clone <repo-url>
cd flashycardy
pnpm install
```

## 2. Environment variables

Create `.env` at the **repository root** (see [Environment variables](/developers/environment-variables)):

```env
DATABASE_URL=postgresql://user:password@host/database?sslmode=require
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_test_...
CLERK_SECRET_KEY=sk_test_...
OPENAI_API_KEY=sk-proj-...
```

## 3. Database migrations

```bash
pnpm --filter @flashycardy/web db:generate:dev
pnpm --filter @flashycardy/web db:migrate:dev:programmatic
```

Schema source: `apps/web/src/db/schema/`. See project rule `drizzle-schema-migrations.mdc` for prod track workflow.

## 4. Run apps

| Command | App | URL |
|---------|-----|-----|
| `pnpm dev` | Web | http://localhost:3000 |
| `pnpm dev:docs` | Docs | http://localhost:3001 |
| `pnpm dev:all` | Web + docs | Both ports |
| `pnpm dev:extension` | Extension (WXT) | Load unpacked from `.output/` |
| `pnpm dev:ios` | Opens Xcode | Simulator/device |

Extension and iOS require the **web app running** on `:3000` for live API calls.

## 5. Extension setup (optional)

```bash
cp apps/extension/.env.development.example apps/extension/.env.development
# Edit Clerk keys + VITE_SYNC_HOST=http://localhost:3000
pnpm dev              # terminal 1
pnpm dev:extension    # terminal 2
```

Load unpacked extension: `apps/extension/.output/chrome-mv3-dev`. See [Chrome extension](/developers/platforms/chrome-extension).

## 6. iOS setup (optional)

```bash
cp apps/ios/Config/Secrets.xcconfig.example apps/ios/Config/Secrets.xcconfig
pnpm dev:ios
```

See [iOS app](/developers/platforms/ios).

## Common commands

```bash
pnpm lint:all          # ESLint all packages
pnpm test:unit         # Vitest (web)
pnpm test:e2e          # Playwright (web; starts prod server)
pnpm build:docs        # Verify docs site builds
```
