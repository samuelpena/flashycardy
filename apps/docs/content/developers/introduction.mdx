# Developer guide

Documentation for contributors, integrators, and platform developers working on the Flashycardy monorepo.

## What is Flashycardy?

Flashycardy is a flashcard product with:

- **`apps/web`** — Next.js 16 primary app (UI, REST API, MCP, Postgres)
- **`apps/docs`** — Nextra documentation site (this site)
- **`apps/extension`** — Chrome MV3 side-panel client (WXT + React)
- **`apps/ios`** — SwiftUI native client
- **`packages/*`** — Shared UI, features, API client, and i18n
